// The single-bit instructions provide a mechanism to set, clear, invert, or extract
// a single bit in a register. The bit is specified by its index.

シングルビット命令は、レジスタの単一ビットをセット、クリア、反転、または抽出するメカニズムを提供する。
ビットはインデックスで指定する。
